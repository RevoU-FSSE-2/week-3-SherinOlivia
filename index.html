How Much Do You Remember of Your Past?

Author's Note

Hi Humans! I'm Sherin Olivia, aka Roo.

This project was dedicated to the two games that I currently play whenever I'm free. One of which is my favourite, Path to Nowhere. Hence why the default mode is Path to Nowhere themed.

I'd like to think that this project is a major step up from the previous one. Therefore I'm utterly proud that I'm able to share and show this to you guys!

I won't dally too much, but the main reasons I chose to do a fan project for the games are:

  • I'm a rather rigid logical person, but I find muses in Art, Music, and Lores.
  • The games easily fit the criterias above. Maybe even more~
  • I usually return the love through writings like poems or epiphanies, this project is a rare chance to do it differently!

Hit Me Up if you want to know more~
